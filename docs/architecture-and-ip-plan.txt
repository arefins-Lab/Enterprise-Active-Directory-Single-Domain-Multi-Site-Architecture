# Enterprise AD Multi‑Site Architecture – IP & Network Plan
Domain: mylab.test

## 1. Overview
This document outlines the full network, IP addressing, and site layout used in the multi‑site Active Directory deployment. The design follows a clean, scalable, enterprise‑style structure across Building‑A and Building‑B.

## 2. Subnet Allocation
- Building A – LAN01: 192.168.1.0/24
- Building A – LAN02: 192.168.2.0/24
- Building B – LAN03: 192.168.3.0/24

## 3. Gateway Plan (pfSense)
- LAN01 Gateway: 192.168.1.1
- LAN02 Gateway: 192.168.2.1
- LAN03 Gateway: 192.168.3.1

## 4. Server IP Plan
- DC-SRV01 (Primary DC + DNS): 192.168.1.210
- DHCP-SRV01 (Primary DHCP): 192.168.1.215
- DHCP-SRV02 (Failover Partner): 192.168.1.220
- ADC-SRV01 (Secondary DC + DNS): 192.168.3.210

## 5. Client IP Plan
All clients receive IPs via DHCP:
- Workstations 01–04
- Desktops 01–02

## 6. AD Site Mapping
- SiteA → 192.168.1.0/24 + 192.168.2.0/24
- SiteB → 192.168.3.0/24

## 7. Replication Topology
- DC-SRV01 ↔ ADC-SRV01 (Intersite replication)
- Default schedule (15 min interval)

## 8. Notes
- Entire architecture designed manually from scratch.
- No templates used.
